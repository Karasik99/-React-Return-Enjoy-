# Пет проект Todo List

## Функционал данного проекта

1 Добавление задачи.
2 Удаление задачи.
3 Сортировка задач.
4 Поиск задач.
5 Просмотр текущих задач.
6 Просмотр завершенных задач.

### Запуск проекта

1 перейти в папку с проектом 
2 установить пакеты `npm install`
3 запуск проекта `npm start`

### Технологии

1 React + Hooks
2 Module.scss
3 Mockapi 
4 Axios
